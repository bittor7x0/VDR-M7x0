     Con el  menú "Ordenes" de VDR, podemos ejecutar cualquier comando del sistema que esté definido dentro del archivo "commands.conf". Este menú sólo nos aparecerá si existe un archivo "commands.conf", válido y que además contenga por lo menos un comando definido, al principio del programa. "VDR-NG-EM para el Siemens Gigaset M740 y M750" viene de fábrica con varias órdenes predefinidas en este menú, pero Vd. puede modificar este archivo para añadir las suyas propias.
     
     Esta característica la podemos utilizar para hacer cualquier cosa, como comprobar si tenemos correo nuevo, o mirar la temperatura de la CPU o mostrar su nombre. Sólo se necesita incorporar la definición del comando en el archivo "commands.conf" y asignar una acción a realizar. Normalmente esta acción suele ser un Shell Script o un script en Perl. Cualquier cosa que el comando escriba en la "stdout", cuando finalice su ejecución automáticamente se mostrará en la pantalla. La pantalla utilizará un tipo de fuente "fija" para poder ajustar el formato de salida. Para no perder los mensajes de error que van a la "stderr", las definiciones de los comandos deben redirigir el "stderr" a la "stdout".
     
     ¡ Precaución !
     LOS COMANDOS QUE SE DEFINAN EN EL ARCHIVO commands.conf SE EJECUTARÁN CON EL MISMO USUARIO CON EL QUE SE ESTÁ EJECUTANDO VDR. TENGA MUCHO CUIDADO A LA HORA DE DEFINIR ESTOS COMANDOS Y ASEGURESE DE QUE NO PROVOCAN NINGÚN DAÑO A SU SISTEMA, ESPECÍFICAMENTE SI EJECUTA EL PROGRAMA CON UN USUARIO CON DEMASIADOS PRIVILEGIOS (POR EJEMPLO root).
