Modo multi-velocidad = no
     
     Define el funcionamiento de las teclas "Izquierda" y "Derecha" en el modo de repetición. Si se establece como "no", solo se utilizara una única velocidad, mientras si se establece a "si", habrá tres velocidades, para la búsqueda rápida y lenta, respectivamente.
     
Mostrar modo de reproducción = no

     Activa o desactiva la visualización del modo de reproducción.
     
ID de continuación = 0

     Establece identificadores adicionales para poderlos  utilizar en entornos donde existan varios usuarios, con lo que cada usuario/a pueda disponer de su propio fichero de "resumen/pausa" de cada grabación. Los valores validos van del 0 al 99. Con el valor 0, se creara  "resume.vdr" como nombre de fichero y si se especifica cualquier otro valor, el fichero que se creara se llamara "resume.n.vdr".
